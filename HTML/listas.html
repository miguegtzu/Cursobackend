Listas

Listas ordenadas

las listas ordenadas le dan un orden a los elmentos que estan dentro de la lista

Ejemplo

la etiqueta OL sirve para listas ordenenadas, para agregar elementos a la lista se utiliza la etiquera LI

  1. primer elemento
  2. segundo elemento
  3. tercer elemento
    1. prueba
    2. prueba
    3. cuarto elemento

    Listas desordenadas

    las listas desordenadas no tienen un orden

    EJEMPLO

    • primer elemento
    • segundo elemento
      • primer elemento
      • segundo elemento
    • tercer elemento
